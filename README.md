
# Minimalist AI Portfolio

Un portfolio per professionisti creativi (Filmmaker & Creative Director) focalizzato sull'estetica noir e minimalista. Caratterizzato da un'esperienza utente fluida e funzionalità di editing segrete.

## Caratteristiche principali

- **Video Carousel**: Carosello orizzontale a schermo intero per mostrare i tuoi lavori video.
- **AI Bio Enhancement**: Utilizza Google Gemini per rifinare la tua biografia professionale.
- **Admin Mode Segreta**: Accesso protetto per caricare contenuti senza bisogno di un backend complesso.
- **Zero-Config Persistence**: I dati vengono salvati localmente nel browser (LocalStorage).
- **Design Minimalista**: Tipografia audace e palette cromatica limitata per un impatto visivo massimo.

## Come attivare la Modalità Admin

1. Vai sul sito live.
2. Clicca velocemente **5 volte** nell'angolo in alto a sinistra della pagina (nell'area dove solitamente risiede il logo).
3. Apparirà il tasto **"Admin Mode"** in alto a destra.
4. Una volta attivo, potrai modificare testi, immagini di profilo e caricare nuovi video direttamente dall'interfaccia.

## Tecnologie utilizzate

- React 19
- Google Gemini API (@google/genai)
- Tailwind CSS
- Lucide Icons (per alcuni elementi grafici)

## Installazione Locale

Per clonare e testare il progetto:

